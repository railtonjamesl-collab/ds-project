We can use some evaluation techniques to see how accurate our model is, such as the confusion matrix.

glm.probs = predict(glm.fit ,type ="response")
glm.pred=rep("0",1250)
glm.pred[glm.probs >.5]="1"
table(glm.pred,sampled_test_data$isFraud)
##         
## glm.pred    0    1
##        0 1246    2

We can then compute our accuracy using a simple formula.

accuracy <- table(glm.pred, sampled_test_data$isFraud)
sum(diag(accuracy))/sum(accuracy)
## [1] 0.9369628

What we are seeing is that the values on the diagonal of the confusion matrix are correct predictions, whilst the off diagonal elements are incorrect predictions. We see then that our model has an accuracy of 93%. Since we are using a synthetic data set, this number could be attributed to the fact that the data is inherently nice for that purpose. Even so, the logistic regression model does seem to fit our data quite well.
